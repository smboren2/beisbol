# Objective

Develop a model that would provide daily MLB fantasy projections. Based on DraftKings scoring, total fantasy points for hitters and pitchers break down as such: 

* Hitters
	+ **Single**: 3   
	+ **Double**: 5   
	+ **Triple**: 8   
	+ **Home Run**: 10   
	+ **RBI**: 2   
	+ **Run**: 2   
	+ **Base on Balls**: 2   
	+ **Hit by Pitch**: 2   
	+ **Stolen Base**: 5 Points

* Pitchers    
	+ **Inning Pitched**: 2.25 
  + **Strike Out**: 2 
  + **Win**: 4 
  + **Earned Run Allowed**: -2 
  + **Hits Given Up**: -0.6 
  + **Base on Balls Against**: -0.6 
  + **Hit by Pitch**: -0.6 
  + **Complete Game**: 2.5 
  + **Shutout**: 2.5 
  + **No Hitter**: 5 Points
